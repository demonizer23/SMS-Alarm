# 🚨 SMS Alarm

## [📲 СКАЧАТЬ ГОТОВЫЙ APK (v1.1)](https://github.com/demonizer23/SMS-Alarm/releases/download/untagged-d08b2900a556c918856e/siren.apk)

> **Внимание:** Для работы приложения на Android 10+ необходимо вручную разрешить "Отображение поверх других окон" в настройках смартфона.
>
Приложение для Android, разработанное на B4A (Basic4android), которое превращает ваш смартфон в сирену. Оно реагирует на входящие SMS-сообщения с кодовыми словами и включает громкий звуковой сигнал, даже если телефон находится в беззвучном режиме или заблокирован.
🌟 Основные возможности
Интеллектуальный фильтр: Реакция только на выбранные номера телефонов и конкретные ключевые слова (настраиваются через запятую).
Максимальная громкость: При срабатывании приложение автоматически выкручивает громкость мультимедиа на 100%.
Работа в фоне: Перехват SMS происходит через фоновый сервис, приложение не обязательно держать открытым.
Обход блокировки: Автоматическое включение экрана и вывод интерфейса поверх LockScreen при поступлении сигнала тревоги.
Удобное управление: Две вкладки — "Пульт" для быстрого теста/остановки и "Настройки" для конфигурации.
Быстрое отключение: Сирену можно выключить экранной кнопкой или любой физической кнопкой смартфона (громкость, назад).
Стильный интерфейс: Скругленные кнопки, кастомный фон и поддержка анимированных GIF.

Вкладка "Настройки": Список реальных отправителей из вашей истории SMS с возможностью выбора (помечаются ✅).
🛠 Технические требования
Необходимые библиотеки B4A:
Для успешной компиляции проекта убедитесь, что в IDE подключены:
Phone (работа с SMS, громкостью и состоянием экрана)
RuntimePermissions (запрос разрешений в реальном времени)
Audio (библиотека для MediaPlayer)
JavaObject (для продвинутой настройки окон и масштабирования графики)
Разрешения Android:
Приложению требуются следующие права:
RECEIVE_SMS и READ_SMS — для анализа входящих команд.
WAKE_LOCK — для включения экрана.
SYSTEM_ALERT_WINDOW — для отображения поверх других окон (нужно включить вручную в настройках телефона).
🚀 Как запустить
Скачайте репозиторий и откройте файл siren.b4a в B4A (v13.10+).
Добавьте свои файлы alarm.mp3, image.gif и imagemain.jpg в папку Files.
Скомпилируйте и установите APK на смартфон.
Важно: В настройках Android найдите это приложение и разрешите ему "Отображение поверх других окон".
